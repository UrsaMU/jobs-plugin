+REQUEST/EXAMPLES

See also: +help request (overview)

EXAMPLES
  +request                          List all your open requests.
  +requests                         Alias — same as above.
  +myjobs                           Alias — same as above.
  +request 3                        View request #3.
  +request My Issue=Please help.    Submit to the default bucket.
  +request/create SPHERE/Bug=Text   Submit to the SPHERE bucket.
  +request/comment 3=Thanks!        Comment on request #3.
  +request/cancel 3                 Cancel your own request #3.
  +request/addplayer 3=Alice        Add Alice as a viewer on #3.

SEE ALSO: +help request, +help request/syntax
